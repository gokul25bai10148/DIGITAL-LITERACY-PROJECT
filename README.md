PROJECT OVERVIEW:

The aim of this project is to investigate the impact that student digital ambassadors have in promoting the digital literacy cause. The student digital ambassadors not only spread the word, but also advocate for the need for a safe and ethical internet by highlighting the critical pillars of internet safety, digital citizenship, and virtual communications. The project also covers the various challenges that come with the cyber world and gives a roadmap for navigating the cyber world with confidence.
